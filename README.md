# kryptonite
AES and DES encryption tool, written for uni. This program aims to encrypt and decrypt a text file (secret.txt) and compare the speed difference between AES and DES. The Python code can be used for other applications as it is written to be used with other software.

# dependencies
Ensure that you are running Python 3.8 or newer, install "pycrypto" and "pycryptodome" from pip3. 

# installation
Clone the repository and change into the "build" directory. Simply run `python3 aes_main.py` or "python3 des_main.py".
